## Project Description

Fiszki AI is a web application that enables users to quickly generate flashcards for language learning. Users provide a topic, and the AI creates a set of 20 flashcards (Polish → English). Users can review, edit, and learn flashcards in interactive sessions.

The application solves the time-consuming process of creating flashcards by offering fast and automatic generation of word sets on a given topic.

### Target Users
- Self-learners of foreign languages
- Students and pupils

## Getting Started Locally

Follow these steps to set up the project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/genaicodedamian/10x-cards.git
   cd fiszki-ai
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:4321`

## Project Scope

### MVP Features
- **Flashcard Generation:**
  - User provides flashcard topic
  - AI generates a set of 20 flashcards: "front" - word in Polish, "back" - translation in English
- **Flashcard Review:**
  - User reviews generated flashcards
  - User accepts or deletes the entire set of flashcards
  - User can edit the "front" and "back" of each flashcard
- **Flashcard Stack Management:**
  - Flashcards are stored in stacks (max 20 flashcards per stack)
  - User can create multiple flashcard stacks
  - The main screen displays a list of flashcard stacks (with generic yellow sticky-note graphics)
  - User can view, edit, and delete flashcards
- **Learning Sessions:**
  - User selects a flashcard stack
  - User reviews flashcards in sessions:
    - The "front" of the flashcard is displayed
    - After clicking on the flashcard, the "back" is displayed
    - User clicks the green button (memorized) or the red button (needs review)
    - After clicking the green button, the next flashcard is displayed
    - After clicking the red button, the flashcard goes to the end of the session list
    - After memorizing all flashcards in the set, the session ends and a congratulatory pop-up is displayed
- **Database:**
  - Storing flashcard data (front, back, creation date, last session date)

### Features Outside MVP Scope
- Custom, advanced repetition algorithm (like SuperMemo, Anki)
- Import of multiple formats (PDF, DOCX, etc.)
- Sharing flashcard sets between users
- Integration with other educational platforms
- Mobile applications
